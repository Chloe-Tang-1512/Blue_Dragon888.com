# Blue_Dragon888.com

This is my first website... I hope you like it!

## 🎮 Games

This site includes three fun, browser-based games:

- **Number Guessing Game** 🔢 — Guess a random number between 1-100 and try to beat your high score
- **Word Scramble Game** 🔤 — Unscramble words with hints and advance through levels
- **Hangman** 🎯 — Classic hangman game with programming-themed words

## 📁 Project Structure

```
main/
├── all the icon files   # Needed for web icon
├── index.html           # Main portfolio page
├── game.html            # Number Guessing Game
├── hangman.html         # Hangman Game
├── wordScramble.html    # Word Scramble Game
├── hmwk0.css            # Stylesheet for all pages
├── hmwk0.js             # Logic for Number Guessing Game
├── hangman.js           # Logic for Hangman Game
├── wordScramble.js      # Logic for Word Scramble Game
└── README.md            # This file
```

## 🎨 Features

- **Responsive Design** — Works on desktop, tablet, and mobile
- **Dark Theme** — Eye-friendly dark blue color scheme
- **Progressive Web App** — Can be installed as a standalone app
- **Game Persistence** — High scores are tracked during your session
